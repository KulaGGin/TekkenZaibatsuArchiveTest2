Tekken Tag Tournament 2/System/Generic Tag Throw List

From Tekken Zaibatsu Wiki
Jump to: navigation, search

Generic Tag Throws can be accessed by pressing 2+5. All Generic Tag Throws do 35 damage and are broken by pressing 2. The animation of the tag throws will depend on which team you have. There are 13 generic tag throw animations in the game and each tag throw gives different Okizeme situations and some breaks floors (which gives 10 extra damage plus bound).

Note: this is not a list for special tag throws, for a more detailed list for special tag throws, go HERE.

(Note: this names are not official)

  • Double DDT (breaks Floors)
  • Double Suplex (breaks Floors)
  • Double Rock Bottom (breaks Floors)
  • Death Drop (breaks Floors)
  • Suplex~Leap Kick
  • Suplex~Lariat
  • Run-over Stomp~Tackle
  • Brawler Combo
  • Flash Kicks Combo
  • Roundhouse~Slide Combo
  • Camel Clutch~Kick
  • Hammer Down Toss
  • Spin Kick Toss
