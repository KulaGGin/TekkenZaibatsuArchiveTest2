Tag Throws

Basic Tag Throw

Basic tag throws can be done by every character with the exception of Devil and Kazuya when they are teamed up together. The initial throw is a Over The Shoulder Slam, the second part of the depends on the character tagging coming into the screen. Only the initial throw can be escaped.

Special Tag Throws

Various characters have special tag throws which to not require any specific team partner. These throws are done by executing a tag right after the throw command. If the throw misses or gets escaped the tag input is ignored and nothing happens. View the character move lists to see which throws allow tagging.
